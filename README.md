# Edward Ijeruh - Portfolio Website

A modern, responsive personal portfolio website built with **React**, **Next.js**, and **Tailwind CSS**.  
This project showcases my work, skills, and experience with a clean UI and smooth user experience.

---

## Features

- **Responsive Navigation**
  - Desktop: Logo + Name, centered nav links, and a “Hire Me” button.
  - Mobile: Animated hamburger menu that slides in from the right.
- **Projects Section** – Displays featured projects with images and descriptions.
- **About Me Section** – Highlights my background, tools, and skills.
- **Contact Section** – Includes links and a way to reach out easily.
- **Modern UI/UX**
  - Smooth animations
  - Optimized images
  - Clean layout and spacing

---

## Tech Stack

- **Framework:** React
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Deployment:** Netlify
